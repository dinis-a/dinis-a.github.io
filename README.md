# Personal Website

This repository contains the source code for my personal website. It's built with a combination of SCSS, JavaScript, and HTML. The goal of the website is to showcase my work, projects, and skills.

## Overview

The website is structured to be clean, responsive, and user-friendly. It highlights my professional background, experience, and includes sections for portfolio projects and contact information.
